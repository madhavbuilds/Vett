============================
  VETT - HOW TO RUN
============================

STEP 1: Open terminal in this folder (D:\vett\vett)

STEP 2: Install dependencies
  pip install -U vett

  (for local development)
  pip install anthropic click questionary rich
  pip install -e .

STEP 3: Get free API key
  Go to: https://console.anthropic.com
  Sign up → Create API Key → Copy it

STEP 4: Enable UTF-8 on Windows (recommended)
  PowerShell: $env:PYTHONUTF8=1
  CMD:        set PYTHONUTF8=1

STEP 5: Set your API key (optional for AI mode)
  Anthropic:  ANTHROPIC_API_KEY
  OpenAI:     OPENAI_API_KEY
  Gemini:     GEMINI_API_KEY
  OpenRouter: OPENROUTER_API_KEY

STEP 6: Run Vett
  vett scan .                  (interactive mode: arrow keys select AI or no-AI)
  vett scan . --no-ai          (scan without API key)
  vett scan . --provider openrouter
  vett scan . --provider openai --model gpt-4o-mini
  vett scan C:\path\to\project (scan any folder)
  vett --help                  (show all commands)
  vett scan --help             (scan command options)
  vett version                 (tool and environment info)

STEP 7: See results
  - Terminal shows colorful report
  - vett_report.md saved in scanned folder

============================
  TROUBLESHOOTING
============================

Error: "vett: command not found"
  Solution: pip install -U vett
  Then reopen terminal and run: vett scan . --no-ai

Error: "ModuleNotFoundError"
  Solution: pip install -U vett

Error: "UnicodeEncodeError" on Windows
  Cause: terminal encoding cannot print emoji
  Solution: set PYTHONUTF8=1, then run again

Error: "No API key found"
  Solution: set ANTHROPIC_API_KEY or use --no-ai

Error: "No source files found"
  Solution: Make sure path has .py .js .ts files
